🚀 Employee Management System (EMS)








A modern web-based Employee Management System that helps manage employee records, attendance, and leave requests efficiently.

The application is designed with two dashboards:

👨‍💼 Admin Dashboard

👨‍💻 Employee Dashboard

This project demonstrates frontend development skills, dashboard UI design, and JavaScript-based data management using LocalStorage.

📌 Project Overview

The Employee Management System (EMS) allows administrators to manage employee information while enabling employees to manage their attendance and leave requests.

The project simulates a real-world company management system where:

Admins control employee data

Employees interact with their own dashboard

Leave requests are handled dynamically

Attendance is tracked through check-in and check-out

✨ Features
👨‍💼 Admin Dashboard

View all employee records

Add new employees

Edit employee details

Delete employee records

View employee attendance

Approve or reject leave requests

Manage overall employee information

👨‍💻 Employee Dashboard

Check-in / Check-out attendance

Submit leave requests

View leave request status

Track attendance history

View personal employee details
